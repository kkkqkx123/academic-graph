生成展示柱状图的前端代码，要求用户可以自定义以下内容：

1. 坐标轴
- 横纵轴可以自己选择数值与字符串2中类型，
- 可以自己设置横纵坐标名称、单位、显示范围、是否使用小数/百分数/分数

2. 图表
- 可以指定柱状图形的个数
- 可以指定每个图形的名称
- 可以选择柱状图是否显示数值，数值可以选择显示在柱状图形上方、中间

3. 样式
- 给出多种风格的柱状图图形作为备选项。要求包含多种计算机或人工智能相关顶刊论文中常用或较出色的样式
- 图表的背景不能使用任何样式，保持空白
- 页面背景随意

4. 数据导入
- 支持通过csv文件快速导入数据
- 支持通过xml文件快速导入样式选项
- 提供配置文件，用户可以通过修改配置文件决定柱状图的样式。启动时读取配置文件
- 如果用户未提供配置文件，可以使用默认的配置

5. 数据导出
- 支持以svg、png格式导出图表

6. 渲染
- 给出"渲染"按钮
- 只有当用户点击开始时才渲染

**其他要求**
- 前端中所有支持的选项都要给出必要说明
- 所有说明使用中文
- 采用模块化设计，并确保代码结构清晰。保证可扩展性，能够在后续添加更多图表样式，能够在后续添加对折线图等图表的支持
- 使用node.js开发简单的后端服务，所有参数设置、样式设置由后端处理。

配置文件格式：
{
  "chart": {
    "type": "bar",
    "title": "示例柱状图",
    "xAxis": {"label": "X轴", "unit": "", "range": [0, 100], "useDecimal": true},
    "yAxis": {"label": "Y轴", "unit": "个", "range": [0, 50], "usePercent": false},
    "showValue": true,
    "valuePosition": "top"
  },
  "style": {
    "theme": "default",
    "backgroundColor": "transparent"
  }
}

数据导入格式：
CSV格式：第一行作为列名，后续行为数据值。 示例：
名称,值1,值2
A,10,20
B,20,30
C,30,40

XML格式：用于定义样式选项。 示例：
<styles>
  <theme>modern</theme>
  <background-color>transparent</background-color>
</styles>
